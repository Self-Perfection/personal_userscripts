# Инструкции для Claude Code

## Перед началом работы

**ОБЯЗАТЕЛЬНО** прочитай файл `AGENTS.md` в корне репозитория. Он содержит важные инструкции для работы с userscripts в этом проекте.

## Ключевые моменты из AGENTS.md

- При создании нового userscript имя должно заканчиваться на `.user.js`
- При обновлении **обязательно** поднять `@version`
- При обновлении **обязательно** добавить `@changelog`
- При пуше в новую ветку **обязательно** показать пользователю ссылку для установки скрипта из этой ветки

## Workflow

1. Прочитай `AGENTS.md`
2. Выполни задачу согласно инструкциям
3. Проверь, что все требования из `AGENTS.md` выполнены
